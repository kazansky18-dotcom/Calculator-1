단위환산 계산기

구성 순서: 1) 질량농도환산 → 2) DustTrack 팩터 → 3) 가스농도(vol%/ppm/mg/m³) → 4) 유량 단위환산 → 5) Actual↔Normal(P,T) + 유량계산(Q=A×V, ṁ=ρ×Q)

1. 질량농도
2. DustTrack Factor
3. Gas: vol% / ppm / mg/m³
4. Flow Units
5. Actual↔Normal + Q=A×V

1. 질량농도 환산 mg/m³ ↔ g/m³ ↔ µg/m³

-
-
-
빠른 확인: 1 g/m³ = 1000 mg/m³ = 1,000,000 µg/m³

2. DustTrack 중량 팩터 계산기 엑셀 수식 동일

필터 전/후 무게와 유량/시간으로 기준 농도를 계산하고, DustTrack 평균값 대비 보정 팩터(New Cal Factor)를 산출합니다.
-
-
-
-
-
Ref(mg/m³)= (Post−Pre) / ((Flow L/min ÷ 1000) × Time min) · NewFactor = (Ref / DustTrackAvg) × CurrentFactor

3. 가스 농도 환산 vol% / ppm / mg/m³

가스 선택 → 분자량 자동 입력. 입력 단위를 선택하면 ppm/vol%/mg/m³가 동시에 계산됩니다.
“직접입력” 선택 시 수정 가능
-
-
-
vol% = ppm / 10,000 · mg/m³ = ppm × MW / Vm (Vm: 몰부피 L/mol, 선택한 표준조건 기준)

4. 유량 단위 환산 m³/h ↔ L/min ↔ Nm³/h ↔ NL/min

-
-
-
-
이 카드의 Nm³ 표시는 “단위 스케일” 환산입니다. 실제 운전조건(P,T) 보정은 5번에서 계산하세요.

5. Actual ↔ Normal 환산 (운전조건 P,T) m³/h ↔ Nm³/h

게이지압이면 절대압으로 변환(예: +101.325)
-
-
이상기체 기반: Qn = Qa × (Pa/Pn) × (TnK/TaK) (K=℃+273.15)

유량 계산 Q = A×V, ṁ = ρ×Q

관/덕트 단면적(A)과 평균 유속(V)로 체적유량(Q)을 계산하고, 밀도(ρ)로 질량유량(ṁ)을 계산합니다.
공기(20℃)≈1.204
